【最初に】
VS CodeのUTF-8環境で開発しました。
Shift JIS環境では文字化けが起こる可能性があります。
Live Serverでローカルwebサーバを立ててゲームを行いました。


【操作方法】
a,dキー:独楽が時計回り・反時計回りに回転 
sキー:独楽リセット

キーを長押しすると、独楽が綺麗に回転しません。

【実装した処理】
生成される独楽の見た目がそれぞれ異なるようにランダム抽選
独楽のY座標が-100より下の時に場外と判定され、オブジェクトの消去と再生成が行われます。
中央の土台の上で反転した場合も同じく再生成が行われます。(438行目のdemoMake関数のコメントアウトを外すことで確認可能)
